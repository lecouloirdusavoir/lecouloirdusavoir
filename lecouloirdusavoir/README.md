# Le Couloir du Savoir

Site statique déployable sur GitHub Pages.

## Structure

```
lecouloirdusavoir/
├── index.html          # Page d'accueil
├── css/style.css       # Feuille de styles
├── js/main.js          # Scripts
└── articles/           # Articles (14 fichiers HTML)
```

## Déploiement sur GitHub Pages

1. Créer un dépôt GitHub nommé `lecouloirdusavoir`
2. Uploader tous ces fichiers à la racine du dépôt
3. Aller dans Settings → Pages → Source : `main` branch, dossier `/ (root)`
4. Votre site sera disponible à l'adresse de votre domaine personnalisé

## Domaine personnalisé (lecouloirdusavoir.com)

Dans Settings → Pages → Custom domain : entrer `lecouloirdusavoir.com`
Créer un fichier `CNAME` à la racine contenant : `lecouloirdusavoir.com`
